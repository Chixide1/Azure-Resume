Fileshare Webapp

Built a file sharing web app using Azure. Terraform was used to build the resources within Azure. Users can upload files with Blob Storage as the backend & generate time-limited download links for sharing. Django & Python handle the frontend & backend. The UI is built with Bootstrap, CSS & HTML.
